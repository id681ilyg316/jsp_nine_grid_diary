## 本项目实现的最终作用是基于JSP九宫格日记本
## 分为1个角色
### 第1个角色为用户角色，实现了如下功能：
 - 个人信息修改
 - 主页
 - 写日记
 - 按照时间线查看日记
 - 用户注册
 - 用户登录
